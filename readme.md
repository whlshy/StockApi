# I3S_API

套裝專案

* I3S_Database
* **I3S_API**

**半套效果：**

* Swagger API文件
* Error Log輸出.txt (可包含API參數)
* ~~Model驗證與客製化回傳~~ (未來規劃有需要的話)

**整裝效果：**

* 完整OAuth 2.0、wke sso登入 (會員全自動插入)
* Permission完整權限驗證機制
* Public API
* Tx API

## 專案版本
.NET 8

## 開啟專案

安裝**Visual Studio**後，點擊**I3S_API.sln**開啟專案
